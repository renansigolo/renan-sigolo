# Renan Sigolo

My Personal Website.

## Build the project for

- Development

```bash
npm run build
``` 

- Production

```bash
npm run build:prod
``` 

### Deploying to production

To deploy a new version to production run one of the following commands

```bash
npm version patch
npm version minor
npm version major
```
